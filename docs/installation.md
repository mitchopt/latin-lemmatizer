# Installation

## Stable release

To install Latin Lemmatizer, run this command in your terminal:

```sh
uv add latin-lemmatizer
```

Or if you prefer to use `pip`:

```sh
pip install latin-lemmatizer
```

## From source

The source files for Latin Lemmatizer can be downloaded from the [Github repo](https://github.com/mitchopt/latin-lemmatizer).

You can either clone the public repository:

```sh
git clone https://github.com/mitchopt/latin-lemmatizer
```

Or download the [tarball](https://github.com/mitchopt/latin-lemmatizer/tarball/main):

```sh
curl -OJL https://github.com/mitchopt/latin-lemmatizer/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd latin-lemmatizer
uv sync
```
